# Starfield Quest - Project Specification

## Project Overview

An interactive starfield animation simulating a spaceship traveling through space. Features a canvas-based star rendering system with adjustable controls for trail length, star count, warp speed, and field radius.

## Project Structure

```
starfield-quest/
├── index.html    # Main HTML structure with control panel
├── script.js     # Star class and animation logic
├── style.css     # Cockpit and control panel styling
└── spec.md       # This file
```

## Commands Used to Create Project

### Initialize Git Repository

```bash
git init
```

### Stage Project Files

```bash
git add index.html script.js style.css
```

### Create Initial Commit

```bash
git commit -m "Initial commit: starfield quest homework assignment"
```

### Add GitHub Remote

```bash
git remote add origin git@github.com:sydneyhamilton6/starfield-quest.git
```

### Push to GitHub

```bash
git push -u origin main
```
